# IntroCard Website

Static website for IntroCard, ready for GitHub Pages.

## Deploy

1. Upload all files and folders to the `introcard` GitHub repository.
2. Go to Settings > Pages.
3. Select `Deploy from a branch`, branch `main`, folder `/root`.
4. Set custom domain to `introcard.app`.
5. Configure DNS with your domain registrar.
6. Enable Enforce HTTPS after DNS has propagated.

## Replace images

Replace the files in `/images` with final app screenshots when ready.
Keep the same filenames or update the HTML references.

Placeholder screenshots (`screenshot1.png` – `screenshot4.png`) currently
stand in for: profile page, vCard QR code, print-ready card design, and
editing card details.
